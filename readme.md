Game (EN) : 2048 / Jeu(FR) : 2048

Description(EN) :

 2048 is a single-player sliding tile puzzle video game written by Italian web developer Gabriele Cirulli and published on GitHub.[2] The objective of the game is to slide numbered tiles on a grid to combine them to create a tile with the number 2048; however, one can continue to play the game after reaching the goal, creating tiles with larger numbers.If all the grids are fully filled and no addition between two identical numbers can be done , the game is over.




Description(FR) : 

2048 est un jeu vidéo de puzzle à tuiles coulissantes pour un joueur écrit par le développeur Web italien Gabriele Cirulli et publié sur GitHub.[2] L'objectif du jeu est de faire glisser des tuiles numérotées sur une grille pour les combiner afin de créer une tuile portant le numéro 2048 ; cependant, on peut continuer à jouer au jeu après avoir atteint l'objectif, en créant des tuiles avec des nombres plus grands. Si toutes les grilles sont entièrement remplies et qu'aucune addition entre deux nombres identiques ne peut être effectuée, le jeu est terminé.



Installation(EN) :

To run this game on a computer , you will need to install python and to install these modules tkinter,random,os.path
In order to install these modules you will need to open your command prompt and press pip install tkinter for example , if you are using visual studio code you can easily download these by clicking on extensions and searching it.




Installation(FR) :

Pour exécuter ce jeu sur un ordinateur, vous devrez installer Python et les modules tkinter, random et os.path.
Pour installer ces modules, ouvrez l'invite de commande et appuyez sur pip install tkinter. Par exemple, si vous utilisez Visual Studio Code, vous pouvez facilement les télécharger en cliquant sur « Extensions ».
